# 技能系统文档

## 品质体系

| 品质 | 颜色 | 最大阶数 | 权重 |
|------|------|---------|------|

## 触发类型

| 类型 | 触发时机 | 
|------|---------|
| PROJECTILE_MODIFIER | 玩家发射子弹时 |
| ON_HIT | 子弹命中敌人时 |
| ON_KILL | 击杀敌人时 |
| ON_CRIT | 暴击时 |
| ON_DASH | 冲刺时 |
| ON_DAMAGED | 玩家受伤时 |
| PERIODIC | 每帧 |
| SUMMON | 每帧（召唤物管理）|

## 游戏机制

### 灼烧
- 每层灼烧每秒造成 `bulletDamage × dps` 伤害
- 可叠加，受刷新持续
- 视觉：敌人变橙红+火焰粒子

### 减速/冻结
- 减速：降低敌人移动速度百分比
- 冻结：完全停止移动
- 视觉：减速变浅蓝，冻结变冰蓝

### 火焰区域
- 圆形地面区域，站上去每秒受伤+叠灼烧
- 受焦土技能强化
- 来自：燃烧地面、火焰风暴
