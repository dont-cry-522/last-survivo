# 末日幸存者 Last Survivor

一款俯视角 2D Roguelike 生存游戏，玩法参考《吸血鬼幸存者》。
纯 HTML5 + JavaScript 实现，无任何第三方依赖，打开即玩。

## 🎮 游戏玩法

### 操作
- **W / A / S / D** - 八方向移动
- **Shift** - 冲刺（2秒冷却，冲刺时无敌）
- **ESC** - 暂停 / 继续
- **R** - 重新开始
- **鼠标点击 / 1 2 3 / 方向键 + Enter** - 选择升级

### 核心循环
1. 飞船自动攻击最近的敌人
2. 击杀敌人掉落经验球
3. 经验满自动升级，三选一强化
4. 敌人随时间越来越强
5. 尽可能存活更久

### 敌人类型
| 类型 | 特点 |
|------|------|
| 🔴 普通敌人 | 基础单位，数量多 |
| 🟡 快速敌人 | 移速快，血量低 |
| 🟣 坦克敌人 | 血厚移速慢，伤害高 |
| 🩷 自爆敌人 | 接近玩家后爆炸造成范围伤害 |
| 🔵 精英怪 | 每30秒出现，高血量高奖励 |
| ❤️ BOSS | 每90秒出现，拥有冲撞和范围AOE技能 |

### 升级选项
- 攻击速度 / 移动速度 / 伤害 / 弹速
- 最大生命 / 紧急修复 / 能量护盾
- 暴击率 / 暴击伤害
- 多重射击 / 穿透强化 / 攻击范围
- 磁力强化 / 经验增益
- 冲刺精通

## 🚀 运行方式

直接用浏览器打开 `index.html` 即可运行，无需任何服务器或依赖。

```
双击 index.html
```

推荐使用 Chrome / Edge 浏览器获得最佳体验。

## 📁 项目结构

```
last-survivor/
├── index.html          # 入口页面
├── style.css           # 页面样式
├── game.js             # 游戏启动入口
└── classes/            # ES6 类模块
    ├── Config.js       # 全局配置（数值平衡、颜色、升级池）
    ├── Utils.js        # 工具函数（数学、碰撞、随机）
    ├── Particle.js     # 粒子系统（对象池）
    ├── Bullet.js       # 子弹系统（追踪、穿透、对象池）
    ├── Enemy.js        # 敌人系统（4种类型+精英）
    ├── Boss.js         # BOSS系统（冲撞、AOE技能）
    ├── Player.js       # 玩家飞船（移动、冲刺、自动攻击）
    ├── Experience.js   # 经验球系统（磁铁效果）
    ├── Upgrade.js      # 升级系统（三选一面板）
    ├── UI.js           # UI渲染（HUD、伤害飘字、结算界面）
    └── Game.js         # 主游戏控制器（循环、相机、碰撞）
```

## ⚡ 性能优化

- **对象池模式**：子弹、敌人、粒子、经验球全部使用对象池复用，避免频繁GC
- **requestAnimationFrame**：流畅60fps游戏循环
- **空间优化**：距离平方比较避免开方运算
- **粒子上限**：最多500个粒子，超出自动覆盖最老的
- **deltaTime 限制**：切屏回来不会大跳
