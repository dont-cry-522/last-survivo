# 🔫 末日幸存者 · Last Survivor

> 俯视角 2D Roguelike 生存射击游戏 | 纯 HTML5 Canvas 实现 | 零依赖 | 打开即玩

[**🕹️ 在线试玩**](https://dont-cry-522.github.io/last-survivo/) · [📦 仓库地址](https://github.com/dont-cry-522/last-survivo)

---

## 🎮 游戏简介

玩家操控一艘科技飞船，在无尽敌人潮中生存。飞船自动瞄准射击，击杀敌人掉落经验球和金币。经验满后升级，从三个随机选项中选择强化。敌人随时间越来越强、越来越快、越来越多，存活越久越上头。

玩法类似《吸血鬼幸存者》，纯 HTML5 + JavaScript + Canvas 实现，**无需安装、无需服务器、无任何第三方依赖**。

---

## 🕹️ 操作说明

| 按键 | 功能 |
|------|------|
| `W` `A` `S` `D` | 八方向移动 |
| `Shift` | 冲刺（2秒冷却，冲刺期间无敌） |
| `ESC` | 暂停 / 继续 |
| `R` | 重新开始 |
| `M` | 静音 / 取消静音 |
| `1` `2` `3` / `方向键` + `Enter` | 升级选择 |
| 鼠标点击 | 选择升级 / 点击按钮 |

## ⚔️ 已实现功能

### 核心玩法
- WASD 八方向移动，Shift 冲刺（带无敌帧，2 秒 CD）
- 自动索敌 + 追踪子弹，支持多重射击和穿透
- 5 种敌人：普通 / 快速 / 坦克 / 自爆 / 精英，各有独特外形
- BOSS 战：冲撞技能（前摇警告线）+ 范围 AOE（红圈预警）
- 难度曲线：敌人越来越多、越来越快、越来越肉

### 成长系统
- 15 种升级选项，三选一，稀有度分级（普通/优秀/稀有）
- 暴击系统、护盾系统、磁铁系统
- 连击计数、金币收集、Boss 击杀统计

### 视觉特效
- 深色科技网格背景，平滑跟随相机无限滚动
- 玩家尾焰粒子、移动残影、冲刺残影
- 子弹追踪拖尾、命中火花、死亡爆炸
- BOSS 出场闪烁警告、震屏效果、红圈 AOE 预警
- 冲撞前摇黄色虚线警告
- 伤害飘字（暴击时金色放大）

### 音效系统
- Web Audio API 程序化生成，无需音频文件
- 射击、冲刺、受击、暴击命中、敌人死亡、自爆
- BOSS 出场/冲撞前摇/冲撞/AOE/死亡
- 升级、拾取、游戏开始/结束、暂停/继续
- 按 M 键一键静音

### UI 系统
- 左上角 HUD：血量护盾、经验条、等级、金币、击杀、时间、波次、连击
- 右上角：FPS、敌人数、子弹数、升级次数
- 底部：冲刺冷却图标（圆形进度+倒计时文字）
- 开始界面、暂停界面、游戏结束统计面板（存活时间/等级/击杀/Boss/最高连杀）

### 性能优化
- 全对象池模式：子弹(200) / 敌人(500) / 粒子(500) / 经验球(300)
- requestAnimationFrame 驱动，deltaTime 限制防切屏大跳
- 距离平方比较避免 sqrt 运算
- 支持 300+ 敌人同屏不卡顿

## 📁 项目结构

```
last-survivor/
├── index.html              # 入口页面（含脚本加载顺序）
├── style.css               # 科技风页面样式
├── game.js                 # 游戏启动入口
├── README.md               # 项目说明文档
└── classes/                # 12 个 ES6 模块类
    ├── Config.js           # 全局数值配置（调平衡全在这里）
    ├── Utils.js            # 数学/碰撞/随机工具函数
    ├── SoundManager.js     # Web Audio API 程序化音效
    ├── Particle.js         # 粒子系统（尾焰/爆炸/命中/残影，500上限）
    ├── Bullet.js           # 追踪子弹（穿透/暴击/拖尾，200上限）
    ├── Enemy.js            # 4 种敌人 + 精英怪 AI
    ├── Boss.js             # BOSS（冲撞前摇/AOE 预警/震屏）
    ├── Player.js           # 玩家飞船（八方向移动/冲刺/自动索敌）
    ├── Experience.js       # 经验球（磁铁吸附/弹跳掉落，300上限）
    ├── Upgrade.js          # 三选一升级面板（稀有度/键鼠操作）
    ├── UI.js               # HUD、伤害飘字、开始/暂停/结算界面
    └── Game.js             # 主循环、相机、碰撞检测、难度曲线
```

## 🎯 设计理念

- 代码结构清晰，模块职责单一，易于扩展
- 流畅的打击感和操作手感
- 足够玩 10 分钟以上的游戏深度
- 纯原生实现，零依赖，性能优越
- 所有数值集中在 Config.js，方便调优平衡
